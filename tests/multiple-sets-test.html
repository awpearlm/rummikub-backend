Multiple Sets Validation Tests

Test Description

These tests verify the functionality of detecting and playing multiple valid sets (specifically groups of tiles) in a single move. We are testing the scenario where a player wants to play two valid groups in one move, such as:

  • Group 1: Blue 4, Yellow 4, Black 4
  • Group 2: Red 7, Blue 7, Yellow 7

The specific issue we're investigating is: "Play selected not recognizing multiple valid groups" where the console shows "Detected 2 valid sets" but then rejects them with "Invalid set rejected by client validation".
